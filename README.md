# INVENTORY-MANAGMENT
It is a simple program usually made for people who needs to maintain the record of something that need updation on daily basis.for example-a shopkeeper need to maintain the record of number of things remaining in the shop as it will help them in maintaining business's profit.

In this program,it allows users to manage product information such as Item ID,Name,Quantity and Price through a menu-driven console interface.
the program supports all essential inventory operations,including:

*Adding new items
*Updating existing items
*Removing items
*Searching for items
*Displaying all items in a clean,colourful table format
